# GAUSS - Simulador TRI

Projeto para simular a correção de provas no modelo TRI (Teoria de Resposta ao Item), utilizado no ENEM. Tendo mais precisão e confiabilidade na hora de simular as notas e sugerir ajustes.

## Objetivo:

Criar uma estrutura capaz de estimar o desempenho de um aluno com base em padrões de resposta, utilizando conceitos de estatística e modelagem de dados. Usando a base de dados fornecida pelo próprio INEP.

## Conceitos aplicados

- Teoria de Resposta ao Item (TRI)
- Probabilidade e estatística
- Modelagem de dados
- Análise de padrões de resposta

## Tecnologias utilizadas

- Python
- Pandas

## Versão atual do projeto

Até aqui, o GAUSS realiza uma simulação estruturada de correção, considerando:

- banco de questões com nível de dificuldade
- gabarito oficial
- respostas do aluno
- cálculo de acertos e erros
- pontuação ponderada por dificuldade
- regra inicial de coerência inspirada na TRI
- classificação simples de desempenho

O objetivo desta fase é construir uma base lógica sólida para a futura evolução do simulador em direção a um modelo probabilístico mais próximo da TRI real.

## Estrutura do projeto

- `notebooks/` → protótipos e simulações iniciais
- `data/` → arquivos de dados utilizados no projeto
- `src/` → código-fonte principal
